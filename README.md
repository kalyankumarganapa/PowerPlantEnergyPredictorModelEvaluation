# ModelEvaluation
Power Plant Energy Output Estimation | Python, Scikit-Learn

Conducted a comparative analysis of Multiple Linear, Polynomial, SVM, Decision Tree, and Random Forest regressors to predict Net Hourly Electrical Energy.

Evaluated model performance using $R^2$ scores across environmental features including Ambient Temperature (AT), Vacuum (V), Ambient Pressure (AP), and Relative Humidity (RH).

Achieved a robust R quare score of 0.94 with the Random Forest model, demonstrating high precision in capturing non-linear relationships within the sensor data.
